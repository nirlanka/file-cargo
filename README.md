coffee-site
===========

Base template for Coffeescript server and front-end

## Starting server

### Prerequisites

Install NodeJS and Git.

Then run:

```bash
npm i -g coffeescript
```

### Download source

Run:

```bash
git clone https://github.com/NirmalL/coffee-site.git
cd coffee-script
npm i
```

### Start server

Run:

```bash
npm start
```
